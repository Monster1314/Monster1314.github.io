正则表达式

正则表达式，又称规则表达式。英文：Regular Expression，在代码中常简写为regex、regexp或re。正则表达式通常被用来检索、替换那些符合某个模式（规则）的文本。

常见场景

  1. 检查字符串合法性

    • 验证用户名
    • 验证邮箱格式
    • 验证电话号码
    • 验证身份证
    • 验证QQ号码格式

  2. 提取字符串中信息

    • 提取一条短信中的数字
    • 提取文件名后缀
    • 采集器

  3. 替换字符串

    • 替换字符串中的非法字符
    • 对电话号码进行屏蔽
    • 替换占位符

  4. 分割字符串

    • 将一个字符串按制定规则进行分割

元字符

使用元字符匹配单个字符

字符功能
.匹配任意一个字符，除了\n
[]匹配[]中列举的字符
\d匹配数字
\D匹配非数字
\s匹配空白，即空格、tab键、换行
\S匹配非空白（数字、英文字符、特殊符号）
\w匹配单词字符，即a-z、A-Z、0-9、_
\W匹配非单词字符
*匹配前一个字符出现0次或者无限次
+匹配前一个字符出现1次或无限次

数量词

使用数量词匹配多个字符

字符功能
{m}匹配前一个字符m次
{m,n}匹配前一个字符m到n次
